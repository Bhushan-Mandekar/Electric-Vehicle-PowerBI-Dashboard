# Electric-Vehicle-PowerBI-Dashboard
Power BI dashboard analyzing electric vehicle population data.

## Project Overview

This project analyzes the Electric Vehicle (EV) population dataset using Power BI to understand EV adoption trends, vehicle distribution, and manufacturer dominance.

The dashboard provides insights into electric vehicle growth, geographical distribution, vehicle types, and electric range categories.

---

## Tools Used

- Power BI
- Data Cleaning (Power Query)
- DAX Measures
- Data Visualization
- GitHub

---

## Data Cleaning Steps

The following transformations were applied:

• Renamed Electric Vehicle Types  
• Removed records with blank vehicle locations  
• Created Electric Range Bin (Low, Medium, High)  
• Extracted latitude and longitude from vehicle location column  

---

## Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

• Total Vehicles  
• Average Electric Range  
• Total BEV Vehicles  
• Total PHEV Vehicles  
• Percentage distribution of BEV and PHEV vehicles  

---

## Dashboard Features

### Main Dashboard

Displays high-level insights:

• Total Electric Vehicles  
• EV adoption trend by model year  
• Electric vehicle distribution by state  
• Top 10 EV manufacturers  
• CAFV eligibility distribution  
• Top 10 EV models  

### Detailed Insights Page

Provides deeper analysis:

• Electric Range Distribution  
• Vehicles by City  
• EV Type Distribution  

Navigation buttons allow users to move between the dashboard pages.

---

## Key Insights

• Electric vehicle adoption has grown significantly after 2015.  
• Battery Electric Vehicles dominate the EV market compared to hybrid vehicles.  
• Some states show significantly higher EV adoption rates.  
• A few manufacturers dominate the EV market.
